Arrays vs. Hashes

Both are used to store and collect data. Arrays are like lists: a collection of data in some order. To access the values in an array, you use something called an index number, which starts at 0:

So if you called fruits[0], you would get strawberry, and so forth.

A hash, on the other hand, is a collection of key-value pairs:

The key red corresponds to the value of strawberry. In other words, if you called fruits[red], you would get strawberry.

A hash is useful if you need to map one value to another, and you can even use arrays for key/value pairs.
