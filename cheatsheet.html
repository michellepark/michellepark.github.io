Git Basics

git branch:
list all branches

git checkout <branch>:
switches between branches

git clone <git://github.com/user/repo.git>:
copies an existing Git repository

git remote -v:
list the current remotes

git fetch:
grabs the remote branch

git status:
state of any changed file(s)

git add:
stages the file to be committed

git commit:
commits the file

git push origin master:
pushes file to remote repo's master

git pull:
automatically fetch and then merge a remote branch into your current branch

git reset HEAD:
brings file back to untracked from tracked

git log:
displays all past commits

git touch :
creates new file

git mkdir <dir_name>:
makes new directory

git cp <filename> <new_filename>:
copies a file to another file

git mv <filename_from> <filename_to>:
moves a file

git rm:
removes a file
